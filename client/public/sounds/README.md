# Door Scanner Sounds

Royalty-free sound files for offline door scanner.

## Files Required

1. **bowling-strike.mp3** - 21+ verified entry (success sound)
   - Suggestion: Bowling strike/pin impact sound (100-200ms)
   - Source: Freesound.org, Pixabay, or Zapsplat (search "bowling strike")

2. **infant-crying.mp3** - Under-21 verified entry (warning sound)
   - Suggestion: Baby/infant crying sound (500-800ms)
   - Source: Freesound.org, Pixabay, or Zapsplat (search "baby cry")

3. **sad-trombone.mp3** - Wrong event type (fail sound)
   - Suggestion: Sad trombone / fail horn (300-400ms)
   - Source: Freesound.org, Pixabay, or Zapsplat (search "sad trombone" or "fail horn")

4. **piano-sting.mp3** - Already-used QR (deny sound)
   - Suggestion: Piano sting / dumdum sound (200-300ms)
   - Source: Freesound.org, Pixabay, or Zapsplat (search "piano sting" or "dumdum")

All files must be:
- MP3 format (44.1kHz, 128kbps minimum)
- Royalty-free (CC0 or CC-BY with attribution)
- Placed in this directory
- Referenced in useDoorSounds.ts
