📋 Day-of Setup Guide

1
At home/hotel (with internet): Go to the admin dashboard → Import tab → click "Download Offline Snapshot". Save the file as bob_snapshot.json.
2
Copy to this laptop: Put bob_snapshot.json in the same folder as START.bat.
3
At the venue: Double-click START.bat to launch this server. Open this page and load the snapshot using the panel above.
4
Create WiFi hotspot: Windows Settings → Network → Mobile Hotspot → Turn On. Note the hotspot name and password.
5
Connect doorman tablets to the hotspot. Open the Network URL shown above on each tablet. Enter the PIN to unlock the scanner.
6
After the event: When internet is available, click "Sync to Cloud Now" to push all redemptions to the cloud database.
